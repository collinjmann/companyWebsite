    Web Developer

    We are seeking a front-end web developer who will be a member of our web team which is responsible for creating, editing, updating, improving, and maintaining quality web content that is accessible through a variety of viewports. Working knowledge of the web stack, HTML5, CSS3, Javascript (Javascript libraries), and Ajax are essential, along with aptitude for backend database integration using PHP and SQL. Ability to build relationships quickly and understand the dynamics of a quick-paced environment is important.

    Roles and Responsibilities

    1. Work closely with Web Team to meet client (internal and external) expectations and deliver exceptional code and consulting advice.
    2. Participate in peer code reviews and project meetings.
    3. Understand client requirements and translate them into technical requirements.
    4. Resolve day-to-day challenges associated with a project.
    5. Provide to project manager time and technical requirements for a project.
    6. Work with team to identify risks with a project.
    7. Contribute to the problem-solving process for project risks and issues.
    8. Utilize all technical resources, both internally and externally, for ensuring technical solutions that represent best practices for modern browsers and various viewports.
    9. Convert requirements into solutions by developing code that follows internal standards and best practices.

    Desirable Technical and Functional Skills

    1. Working knowledge of the trifecta: HTML5, CSS3, and Javascript.
    2. Familiarity with Javascript libraries (Jquery) and Ajax.
    3. Familiarity of backend integration using PHP and SQL.
    4. Knowledge of W#C Accessibility Guidelines and compliance with federal accessibility laws.
    5. Ability to work well with others, inspiring creativity and constructive conflict resolution.
    6. Strong time management and organizational skills.
    7. Ability to work on multiple projects under strict deadlines.
    8. Problem-solving skills that are creative and efficient.
    9. Contributes to a productive, inspiring, and fun work environment.
