    Web Developer

    We are seeking a front-end web developer who will be a member of our web team which is responsible for creating, editing, updating, improving, and maintaining quality web content that is accessible through a variety of viewports. Working knowledge of the web stack, HTML5, CSS3, Javascript (Javascript libraries), and Ajax are essential, along with aptitude for backend database integration using PHP and SQL. Ability to build relationships quickly and understand the dynamics of a quick-paced environment is important.

    Roles and Responsibilities

    1. Work closely with Web Team to meet client (internal and external) expectations and deliver exceptional code and consulting advice.
    2. Participate in peer code reviews and project meetings.
    3. Understand client requirements and translate them into technical requirements.
    4. Resolve day-to-day challenges associated with a project.
    5. Provide to project manager time and technical requirements for a project.
    6. Work with team to identify risks with a project.
    7. Contribute to the problem-solving process for project risks and issues.
    8. Utilize all technical resources, both internally and externally, for ensuring technical solutions that represent best practices for modern browsers and various viewports.
    9. Convert requirements into solutions by developing code that follows internal standards and best practices.

    Desirable Technical and Functional Skills

    1. Working knowledge of the trifecta: HTML5, CSS3, and Javascript.
    2. Familiarity with Javascript libraries (Jquery) and Ajax.
    3. Familiarity of backend integration using PHP and SQL.
    4. Knowledge of W#C Accessibility Guidelines and compliance with federal accessibility laws.
    5. Ability to work well with others, inspiring creativity and constructive conflict resolution.
    6. Strong time management and organizational skills.
    7. Ability to work on multiple projects under strict deadlines.
    8. Problem-solving skills that are creative and efficient.
    9. Contributes to a productive, inspiring, and fun work environment.

  • +  Server Administrator

    Server Administrator

    We are seeking a server administrator who will be a member of our Networking Team. Working knowledge of Microsoft Server 2008 and 2012, with specific emphasis on Active Directory management is very important. Must possess a customer service attitude, work well under pressure, and be a self-driven learner. Productive collaboration with team members is essential for providing responsive, quality solutions to end user problems.

    Roles and Responsibilities

    1. Manage day-to-day administration of Active Directory including: Creating, modifying, removing WIndows security groups, file shares, folders, and file permissions.
    2. Manage the day-to-day enterprise messaging environments including: creating, modifying, or deleting of: mailboxes, distribution lists, mail contacts, and transport rules.
    3. Perform troubleshooting as required.
    4. Ensure high availability and acceptable levels of performance of mission critical host computer resources.
    5. Execute procedures to maintain security and protect systems from unauthorized use, acts of nature, and user abuse.
    6. Execute procedures for the installation and management of the host computer operating systems, disk arrays, ethernet switches, tape libraries, and other components.
    7. Review, test, and deploy security patches to the information systems environment.
    8. Contribute to team efforts by establishing and maintaining professional working relationship with peers and end users.
    9. Inform appropriate members of the Network Team regarding new issues.

    Desirable Technical and Functional Skills

    1. Proficient knowledge of desktop OS, MS Server 2008 and 2012, HyperV, Active Directory, and MS Exchange.
    2. Skill in using PowerShell for administration tasks.
    3. Understand and manage firewalls.
    4. Ability to configure print servers and push out via Group Policy.
    5. Working knowledge of DHCP/DNS management and configuration.
    6. Ability to install software and develop images for distribution according to procedures.
    7. Ability to document, track, and monitor all problems, bringing them to timely resolution with a quality outcome.
    8. Possesses a servant-oriented attitude, including patience and resolve to meet customer demands.
